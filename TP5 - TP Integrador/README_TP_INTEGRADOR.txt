Nombre del script: TP_INTEGRADOR.ipynb
Desarrollada con: Jupyter 
Version: 7.2.2

Descripción:  
- Este script realiza web scraping + spaCy sobre una página web específica para extraer información de interés.
- La URL de trabajo es: https://vueltaolimpica.com/index.php

Requisitos:  
Este script requiere Python 3 y las siguientes librerías:
- requests
- beautifulsoup4
- pandas (opcional, para exportar los datos)
- spaCy
- wordcloud
- matplotlib
- sklearn.feature_extraction.text

Notas:
- Asegúrate de revisar los Términos de Servicio del sitio web antes de hacer scraping.
